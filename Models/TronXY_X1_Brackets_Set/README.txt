# Summary

This is the set of brackets that are from my YouTube review of the TronXY X1 3d Printer.

These brackets MAY have to be altered slightly or the holes made a little larger to fit. I modeled these in a hurry. All the brackets I printed fit on my machine. If you alter the brackets, please submit them back to me so I can update the files here and everyone else can benefit. 

I recommend PLA and more than 60% infill.
