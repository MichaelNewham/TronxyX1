# Summary

Out of the box the Tronxy X1 can be a bit wobbly in the Z axis. 
In my case it would not even want to move upwards. The problem is completely gone now and the rod is held in place nicely.
I also added holes for the handle that came with the printer.

Also included are the FreeCAD files so you can modify the part to your liking.
